NAT

家里 / 公司路由器就是NAT：内网设备（手机 / 电脑）用私网 IP，访问外网时，NAT 会给你分配一个公网 IP + 端口做映射，外网只能看到这个公网映射，看不到你内网真实 IP。

切片

切片底层不是数组而是带数组的结构体
type slice struct {
array unsafe.Pointer // 指向底层数组的指针
len int // 长度：当前元素个数
cap int // 容量：底层数组总可存元素个数
}
扩容规则
原容量 < 1024：容量翻倍
原容量 ≥ 1024：每次增长 1/4

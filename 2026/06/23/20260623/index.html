Java相关

发表于2026-06-23|更新于2026-06-23
|浏览量:

在 Java 中，方法和属性的访问控制修饰符有 private、protected、public 和 default。

设 Point 为已定义的类，创建 Point 对象 a 的语句是：

答案： Point a = new Point();

为类 Point 定义一个没有返回值的无参数方法 move()，如果想通过类名访问该方法，则该方法的声明为：

答案： public static void move()

使用 SimpleDateFormat 类对日期进行格式化，显示 “16 时 30 分 15 秒” 格式的语句为：

答案： SimpleDateFormat sdf = new SimpleDateFormat("HH时mm分ss秒");

设有变量 a 和 b，使用 Math 类中的方法计算 a 的 b 次幂的语句为：

答案： Math.pow(a, b)

假设 s = "Java String"，写出下列操作的输出结果：

操作 输出结果
s.toUpperCase() JAVA STRING
s.toLowerCase() java string
s.indexOf("v") 2
s.replace("ing", "gni") Java Strgni

泛型 <K extends Object> 中，extends 并不代表继承，它是 上限 限制。

Collection 接口中，实现获取集合中元素的个数的方法是：

答案： size()

实现 List 接口的类有 ArrayList 和 LinkedList。

当使用数组中不存在的索引时，会引发 ArrayIndexOutOfBoundsException 异常。

当布尔表达式为 false 时，assert（断言） 语句会触发 AssertionError 异常。

同一段程序可能产生不止一种异常，可以放置多个 catch 子句，一旦代码抛出异常，第一个与之匹配的就会被执行。

抛出异常、生成异常对象都可以通过 throw 语句实现。

如果不想在方法中处理异常，可以在方法声明时通过 throws 关键字将其抛出，交给调用该方法的代码处理。

Java 中 IO 流根据数据传输方向的不同可分为 输入流 和 输出流，根据数据传输单位的不同可分为 字节流 和 字符流。

File 类中：

  • 用于确定文件或目录是否存在的方法是 exists()
  • 用于获取绝对路径的方法是 getAbsolutePath()
  • 用于创建文件的方法是 createNewFile()
  • 用于删除文件或目录的方法是 delete()

在文件的任意位置进行既读又写的操作，应当使用 RandomAccessFile 类。

I/O 流是由 java.io 包来实现的。

线程的生命周期通常需要经历 5 种状态，分别是 新建 (New)、就绪 (Runnable)、运行 (Running)、阻塞 (Blocked) 和 终止 (Terminated)。

Java 中 yield() 方法可让步给高优先级或同等优先级的线程。

FD是什么？
fd 是什么1. 全称File Descriptor，文件描述符 2. 核心一句话Linux 下一切皆文件：普通文件、TCP连接、UDP套接字、管道、终端、socket、定时器，在内核里全部统一当成文件管理，内核给每一个打开的资源分配一个非负整数编号，这个编号就是 fd。 3. 举最简单例子程序启动默认自带 3 个 fd： 0：标准输入 stdin（控制台输入） 1：标准输出 stdout（打印日志） 2：标准错误 stderr（报错信息） 你打开一个文本文件，操作系统返回数字 3，那这个文件的 fd=3；建立一条TCP客户端连接，返回 4，这条网络连接 fd=4。 4. fd 存在在哪？每个进程单独维护一张 文件描述符表： key：fd 数字 value：指向内核全局的文件对象（socket/文件/管道） 不同进程的 fd 互不干扰，进程A的fd=3 和进程B的fd=3 是完全不同的资源。 5. 关键特性（面试必考点） 资源占用，有上限系统通过 ulimit -n 限制单个进程最大fd数量，默认一般 1024。...
